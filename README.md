# Moe Kyaw Aung Portfolio App

⭐ **Android Senior Developer Portfolio** - Built with Kotlin + Jetpack Compose + MVVM + Clean Architecture

## 🎯 Features

- ✅ **Modern UI**: Jetpack Compose with Neon Cyan/Pink/Yellow theme
- ✅ **MVVM Architecture**: Clean separation of concerns
- ✅ **Room Database**: Offline-first data caching
- ✅ **Dark/Light Mode**: System-aware theme toggle
- ✅ **Animated Stats**: Counter animations with Coroutines
- ✅ **Progress Bars**: Skill percentage with smooth transitions
- ✅ **Project Cards**: GitHub integration with demo links
- ✅ **Contact Form**: Email verification flow

## 🛠️ Tech Stack

| Category | Technology |
|----------|------------|
| Language | Kotlin 1.9+ |
| UI | Jetpack Compose + Material 3 |
| Architecture | MVVM + Clean Architecture |
| Database | Room + SQLite |
| Networking | Retrofit + OkHttp |
| Async | Coroutines + Flow |
| Dependency | Hilt (Dagger) |
| CI/CD | GitHub Actions |
